1.git的操作步骤
    1. 初始化目录---目的是让这个目录归git管理
    	git init
    	会在目录下创建一个.git的文件夹
    2. 工作区 -> 缓存区
    	git add . | <file>    .全部
    	git add . 把工作区的文件全部放到缓存区
    	git add <file> 把特定的文件上传到缓存区

    3. 缓存区 -> 版本库

    	git commit -m "日志"

2. git连接github
 		用的是SSh

 	1. 需要生成私钥和秘钥
 		ssh-keygen -t rsa -C "github邮箱"

 		一路回车

 	2. 找私钥和秘钥的位置

 		系统文件 .ssh目录下
 		window 找目录
 		mac 
 			cd ~/.ssh
 			open .
 	3. 公钥和私钥文件
 		id_rsa 私钥文件
 		id_rsa.pub  公钥文件 
 			公钥文件打开，复制里面的内容，给github用

 	4. 过程参考git链接github文件的示意图

 	5. 需要把本地的id_rsa.pub文件的内容给到github的ssh中

 	6. 检验是否链接上github
 		ssh -C git@github.com

 		回车，出现：
 			hi 你的用户名，就成功了


 3. 通过git上传到github上
 	1. 建一个空仓库
 	2. 本地的文件上上传
 			git init
 			git add .
 			git commit -m "日志"
 	3. 连接远程仓库地址
 		git remote add origin '仓库地址'

 	4. 本地的版本库推送到远程
 		 git push -u origin master

 	第二次在push到远程的时候，不需要git remote这个命令了

4. 克隆别人的仓库中的代码到本地
	git clone 仓库地址


5. cmd命令行命令
	cd 进入的路径
	ls 展示这个目录下的文件
	tab 自动补全
	cls 清屏
	ipconfig 查找ip
